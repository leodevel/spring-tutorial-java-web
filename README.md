# Tutorial Spring Framework #

Este exemplo mostra o código fonte utilizado para listar, cadastrar, atualizar e remover usuários de uma lista estática. 

## Frameworks ##

* Spring framework 4.0.1
* JSTL 1.2.1
* Bootstrap 3.3.7
